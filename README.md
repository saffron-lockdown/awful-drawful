# drawful-clone

A clone of Drawful using express, socket.io and Vue.js.

## Getting started

```sh
npm install
npm run dev
```

## Running in production

```sh
npm run build
npm start
```

## Unit tests

To run unit tests:

```sh
npm run test
```

## Integration tests

Tests are located in `cypress/integration`.

To run integration tests:

```sh
npm run test:integration
```
